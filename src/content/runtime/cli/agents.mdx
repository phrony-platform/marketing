## Synopsis

```bash
phrony agents <subcommand> [args] [flags]
```

## Description

Agent lifecycle commands: validate and publish manifests locally or to the runtime, inspect versions, deploy activations, and manage registry state. All subcommands except [`validate`](/docs/runtime/cli/validate) and [`diff`](/docs/runtime/cli/diff) require a running runtime.

| Subcommand | Page | Summary |
|------------|------|---------|
| `agents ls` | — | List agents in the runtime |
| `agents archive` | — | Archive an agent (deprecate all versions) |
| `agents validate` | [`validate`](/docs/runtime/cli/validate) | Validate an agent manifest locally |
| `agents diff` | [`diff`](/docs/runtime/cli/diff) | Diff local manifest against a published version |
| `agents publish` | [`publish`](/docs/runtime/cli/publish) | Publish an immutable agent version |
| `agents versions` | [`versions`](/docs/runtime/cli/versions) | List published versions |
| `agents inspect` | [`inspect`](/docs/runtime/cli/inspect) | Metadata for a published version |
| `agents deprecate` | [`deprecate`](/docs/runtime/cli/deprecate) | Mark a version as not runnable |
| `agents retire` | [`retire`](/docs/runtime/cli/retire) | Retire a published version |
| `agents deploy` | [`deploy`](/docs/runtime/cli/deploy) | Activate a published version for sessions |
| `agents active` | [`active`](/docs/runtime/cli/active) | Show the active deployed version |
| `agents history` | [`history`](/docs/runtime/cli/history) | Deployment activation history |

[`rollback`](/docs/runtime/cli/rollback) stays at the top level for quick access.

## `agents ls`

List agents.

```bash
phrony agents ls [flags]
```

### Flags

| Flag | Short | Description |
|------|-------|-------------|
| `--namespace` | `-n` | Filter by namespace |

### Example

```bash
phrony agents ls
phrony agents ls -n demo
```

### Output

Table columns: `ID`, `NAMESPACE`, `NAME`, `OWNER`, `STATUS` (`active` or `archived`).

---

## `agents archive`

Archive an agent and deprecate **all** of its versions.

```bash
phrony agents archive AGENT
```

`AGENT` is `namespace/name`.

### Example

```bash
phrony agents archive demo/echo-agent
```

### Output

```text
archived agent demo/echo-agent (all versions deprecated)
```

Archived agents cannot be deployed or run.
