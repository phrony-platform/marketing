title: "active"

---

## Synopsis

```bash
phrony active AGENT
```

## Description

Prints the version currently activated for sessions in this runtime — the latest [`deploy`](/docs/runtime/cli/deploy) or [`rollback`](/docs/runtime/cli/rollback) — plus when it was activated and by whom.

`AGENT` is `namespace/name`.

## Example

```bash
phrony active demo/echo-agent
```

Example output:

```text
demo/echo-agent@1.2.0 deployed 2026-05-31T12:00:00Z by ci@github
```
