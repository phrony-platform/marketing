---
title: "status"
description: "Show CLI and runtime status."
---

## Synopsis

```bash
phrony status
```

## Description

Connects to the runtime and prints a status panel: configured runtime address, CLI version, runtime version, schema version, and gRPC health.

## Example

```bash
phrony status
```

Requires a running `phrony-runtime` and a reachable gRPC address (see [CLI overview](/docs/runtime/cli#connect-to-the-runtime)).
