---
title: "inspect"
description: "Show metadata for a published agent version."
---

## Synopsis

```bash
phrony agents inspect AGENT@VERSION
```

## Description

Prints metadata for one published version: content hash, publish time, and optional deprecation or retirement timestamps.

`AGENT@VERSION` is `namespace/name@semver`.

## Example

```bash
phrony agents inspect demo/echo-agent@1.2.0
```
